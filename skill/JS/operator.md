
+ 位运算
```js
&   // 按位与  全1 为1   1001 & 1100  = 1000
|   // 按位或   有1为1   1001 | 1100  = 1101
^   // 按位异或  不同为1 ，相同为0  1001 ^ 1100 = 0101  
~   // 翻转操作位           ~ 1001   = 0110
<<  // 左移 ，右边填零      1001<<2  =  100100
>>  // 右移 ，丢弃移出的位     ，左边填充有符号约束（最高位表示符号，补码表示法）
>>> // 无符号又移   ，全部填0
```
